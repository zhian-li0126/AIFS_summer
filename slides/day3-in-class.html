What is Machine Vision?

Machine vision uses cameras and algorithms to extract useful information from images.

In greenhouses, machine vision can monitor plant size, color, stress, disease symptoms, and growth rate.

Images are Numbers

A grayscale image is a matrix of intensity values.

A color image usually has three channels: red, green, and blue.

Example: A healthy lettuce leaf often has stronger green channel values than red or blue.

Resolution and Field of View

Resolution is the number of pixels in an image.

Field of view is how much of the scene the camera sees.

If a camera has 1280 × 720 pixels, then each image contains 921,600 pixels.

Higher resolution can capture more detail, but also increases file size and processing time.

Preprocessing: Cleaning the Image

Resize

Make images the same size.

Normalize

Scale pixel values consistently.

Enhance

Improve contrast or reduce noise.

Example: Before training an image classifier, all plant images might be resized to 224 × 224 pixels.

Color Spaces: RGB vs HSV

RGB

Stores color as red, green, and blue intensity.

Useful for display and deep learning input.

HSV

Stores hue, saturation, and value.

Often useful for separating green leaves from background.

Simple Green Threshold Example

# Very simple idea, not production code:
if green_value > red_value and green_value > blue_value:
    pixel = "likely leaf"
else:
    pixel = "likely background"
Limitation: Lighting, shadows, camera settings, and leaf color variation can break simple rules.

Common Imaging Problems

Lighting variation

Changes color and contrast.

Occlusion

Leaves overlap each other.

Background clutter

System hardware appears in the image.

Key idea: Good imaging experiments control the camera, lighting, distance, background, and naming scheme.

Greenhouse Applications

  • Track plant growth without destructive harvest.
  • Detect leaf yellowing, wilting, or deficiency symptoms.
  • Estimate canopy area and growth rate.
  • Support automated decision-making in CEA systems.
