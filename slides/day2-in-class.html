Analog vs Digital Signals

Analog

Continuous voltage range.

Example: 0–5 V from a light sensor.

Digital

Discrete values or messages.

Example: DHT sensor sends a digital humidity value.

From Voltage to Meaning

Sensor Value = Calibration Slope × Voltage + Intercept

Example: A light sensor has calibration:

PPFD = 0.8 × millivolts

If the sensor output is 500 mV, then PPFD = 0.8 × 500 = 400 μmol m⁻² s⁻¹.

Electrical Basics

Voltage

Electrical potential difference.

Like pressure in a pipe.

Current

Flow of electric charge.

Like water flow rate.

Resistance

Opposition to current.

Like a narrow pipe.

V = I × R

Ohm’s law connects voltage, current, and resistance.

ADC: Analog-to-Digital Converter

An ADC converts a voltage into a digital number that a microcontroller can process.

Arduino Uno example: 10-bit ADC gives values from 0 to 1023.

If reference voltage is 5 V:

Voltage = ADC × 5.0 / 1023

int raw = analogRead(A0);
float voltage = raw * 5.0 / 1023.0;

Sensor Calibration

Calibration connects sensor output to a real measurement.

Without calibration, a sensor may give numbers but not reliable values.

Two-point calibration example:

Known pH 4 solution → voltage A

Known pH 7 solution → voltage B

Use these points to build a line that converts voltage to pH.

Noise and Sensor Errors

Noise

Random variation in readings.

Drift

Sensor slowly changes over time.

Bias

Sensor is consistently too high or too low.

Example: If temperature readings jump between 22.1°C and 22.8°C every second, averaging may help smooth random noise.

What is IoT?

Internet of Things (IoT) means physical devices collect data and communicate through networks.

Greenhouse IoT example:

A Raspberry Pi reads greenhouse temperature every minute and uploads the data to a dashboard. If temperature exceeds a threshold, it sends an alert.

Interpreting Sensor Data

Scenario: Light increases from morning to noon. Temperature also rises.

Question: Did temperature rise because of light, poor ventilation, or both?

Sensor data becomes useful when we compare multiple variables over time.

Example Control Rule

if air_temp_C > 28:
    turn_fan_on()
elif air_temp_C < 24:
    turn_fan_off()
else:
    keep_current_state()
Important: Real systems need safety checks. A sensor failure should not cause unsafe actuator behavior.

Hands-on Preview

  1. Connect sensor power, ground, and signal wires.
  2. Read values through Arduino or Raspberry Pi.
  3. Print values to Serial Monitor or terminal.
  4. Save the readings as a small dataset.
  5. Discuss whether the readings make physical sense.

Day 2 Key Takeaways

  • Sensors convert greenhouse conditions into measurable signals.
  • Calibration is required to turn raw signals into meaningful units.
  • Arduino is strong for direct sensing and control; Raspberry Pi is strong for networking, cameras, and Python workflows.
  • IoT connects physical systems to data storage, dashboards, and alerts.
